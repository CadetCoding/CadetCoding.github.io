THE STORAGE BOX: Variables

On a long space mission, we need to keep track of our supplies. Instead of typing the same number over and over, we put that number into a Variable.

Think of a variable like a Cardboard Box. You write a name on the outside (like fuel) and put a value inside (like 100).

Name Tag:
fuel
Inside:
100
📦 THE ASSIGNMENT RULE:
In Python, the = sign doesn't mean "equals." It means "Put this in the box!"

fuel = 100 means "Put the number 100 into the box named fuel."
