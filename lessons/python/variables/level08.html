THE UPLINK: input()

To launch the rocket, we need the Pilot's name. We use the input() function to ask a question and wait for an answer.

Whatever the Cadet types back gets stored inside our variable box!

📡 HOW IT WORKS:
name = input("Who is the Pilot?")
This stops the program, waits for you to type, and saves your name into the name box.
