THE JOINER: String Concatenation

Great work on Mission 01! But sometimes, one megaphone isn't enough. You might have two different words that need to be glued together to make a full sentence.

In Python, we use the + symbol. For numbers, it does math. But for words (Strings), it acts like Space-Glue!

⚠️ THE HIDDEN SPACE TRAP:
Python is very literal. If you glue "Rocket" + "Ship", you get "RocketShip".

If you want a space, you have to add it yourself! Either "Rocket " + "Ship" or "Rocket" + " " + "Ship".

🤖 ROBOTICS HUB

Why do robots need "Word Glue"?

  • Sensors: A robot might take the word "Distance: " and glue it to the number from a laser sensor to show "Distance: 10cm".
  • NASA: Mission Control glues "Status: " + "Fuel Low" to create warning alerts for astronauts.
