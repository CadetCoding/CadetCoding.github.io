THE BLUEPRINT: Multi-Line Art

NASA engineers don't just send words; they send blueprints! In Python, you can use multiple print() commands to draw pictures.

🚀 MISSION GOAL:
Build a mini-rocket by printing three lines.

🛠️ COMMAND CONSOLE

Type the blueprint exactly to build the rocket:

print("  ^  ")
print(" | | ")
print("/_ _\\")
print(" ^ ") print(" | | ") print("/_ _\\")
